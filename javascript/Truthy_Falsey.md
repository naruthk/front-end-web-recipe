# Truthy & Falsey

## Falsey

A falsey value is a value that evaluates to `false` when converted to a boolean

- `false`
- `undefined`
- `null`
- `0` (numeric zero)
- `""` (empty string)
- `NaN` (Not a Number)

## Truthy

A truthy value is a value that evaluates to `true` when converted to a boolean

- Any number not `0`
